# Khatastroffik Treefolder

A node.js CLI utility to display a tree representation (treeview) of a file &amp; folder structure.

## Usage (in progress)

&rarr; **Run the typescript code directly**

```shell
pnpm tsx .
```

or

&rarr; **Build and run the javascript code**

```shell
pnpm build
node . # or "pnpm start"
```

or

&rarr; **Run and watch for code changes during development**

```shell
pnpm dev
```
